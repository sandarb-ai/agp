# AIGP Adopters

Organizations and projects that have adopted the AI Governance Proof (AIGP) specification.

If your organization uses AIGP, please submit a pull request to add yourself to this list.

## Adopters

| Organization | Description | Since | Link |
|-------------|-------------|-------|------|
| Sandarb | Reference implementation — AI Governance Platform producing AIGP events across A2A, MCP, REST, and SDK integrations | v0.1.0 | [sandarb.ai](https://sandarb.ai) |

## How to Add Your Organization

1. Fork this repository
2. Add a row to the table above
3. Open a pull request with a brief description of how you use AIGP

We welcome all adopters — whether you produce AIGP events, consume them, or build tooling around the specification.
